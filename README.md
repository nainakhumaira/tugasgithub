## Simple Website

This is a simple static website hosted on GitHub Pages with the following features:
- **Setup**: Instructions on how to set up the website.
- **Biaya**: Information about costs (free hosting).
- **Bandwidth**: Information about bandwidth in web hosting.
- **Backend**: Mock backend functionality using JavaScript, including a contact form.

### How to Deploy
1. Go to repository settings.
2. Enable GitHub Pages from the main branch.
3. The site will be available at `https://nainakhumaira.github.io/tugasgithub/`.
